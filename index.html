Flujo de Trabajo Básico

Cuando se trabaja en Git u otros sistemas de control de versiones, el concepto de “guardar” es un proceso con más matices que el de guardar en un procesador de texto.

Para guardar los cambios, en Git se sigue un proceso distinto. En git hablamos de commit o confirmaciones. Estas pueden capturarse de manera local, y después enviarse a un servidor remoto.
